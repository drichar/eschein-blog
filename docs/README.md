## Contributing

When adding new documentation:

1. Use the date format `YYYY-MM-DD` in the filename
2. Keep titles concise but descriptive
3. Add an entry to this README with a brief summary
4. Include relevant sections: Overview, Changes, Testing, Rollback Instructions
5. Link to relevant external documentation
